# 深色模式颜色层次分析

## 旧版本的完美层次（从深到浅）

### 视觉层次（4层）
```
Layer 4 (最亮): 输入框、搜索框              #283345  rgb(40, 51, 69)
Layer 3 (亮):   侧边栏、卡片内容            #283345  rgb(40, 51, 69)  
Layer 2 (中):   卡片背景、对话框            #1e293b  rgb(30, 41, 59)
Layer 1 (深):   页面背景、主内容区          #0f172a  rgb(15, 23, 42)
```

## 当前新版本的问题

### 现在的配置：
```scss
--el-bg-color: #1e293b;              // 主背景：卡片
--el-bg-color-page: #0f172a;         // 页面背景
--el-bg-color-overlay: #283345;      // 浮层背景
--el-input-bg-color: #283345;        // ✅ 已修复：输入框背景
```

### 问题分析：
1. ✅ 页面背景 `#0f172a` - 正确
2. ✅ 卡片背景 `#1e293b` - 正确  
3. ✅ 浮层背景 `#283345` - 正确
4. ✅ 输入框背景 `#283345` - 已修复
5. ❌ **但是实际渲染时可能被其他样式覆盖了！**

## 需要确认的地方

### 1. 侧边栏背景
应该使用：`#283345` 或 `#1e293b`（取决于设计）

### 2. 表格行背景
- 普通行：透明或 `#1e293b`
- 悬停行：`#283345`

### 3. 搜索栏/筛选区域
应该使用：`#283345`（比卡片更亮）

### 4. 按钮背景
- 次要按钮：`#283345`
- 主要按钮：主题色渐变

## RGB 值对比表

| 层次 | 十六进制 | RGB | 用途 |
|------|----------|-----|------|
| Layer 1 | #0f172a | rgb(15, 23, 42) | 页面背景、最深层 |
| Layer 2 | #1e293b | rgb(30, 41, 59) | 卡片、对话框、模态框 |
| Layer 3 | #283345 | rgb(40, 51, 69) | 侧边栏、输入框、浮层（+10,+10,+10） |
| Layer 4 | #334155 | rgb(51, 65, 85) | 填充色、高亮 |

注意：每层之间的差值约为 10-15 RGB 单位，足够肉眼区分但不会太突兀。
